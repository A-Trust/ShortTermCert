# Short Term Cert
**Short Term Cert** is a qualified short lived certificate 
in accordance with the eIDAS regulation (https://eur-lex.europa.eu/eli/reg/2014/910/oj, 
Section 4, Article 28, Qualified certificates for electronic signatures)

**Short Term Cert** is a remote signature short lived certificate, therefore the private key is stored 
in a hardware security module in the A-Trust data center and is deleted after the session ends or after 
a fixed time intervall of 15 minutes.
